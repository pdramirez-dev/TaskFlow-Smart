# TaskFlow Smart

Marketing site for the TaskFlow Smart iOS app, built with Astro, TypeScript, Tailwind CSS and React components.

## Commands

```bash
npm install
npm run dev
npm run build
npm run check
```

## Routes

- `/`
- `/privacy`
- `/support`
- `/404`
